Necə qoşulur?

HubMSG API V1

API Sənədləri

Baza URL:
http://localhost:2004
Header Parametrləri
Açar Dəyər Mütləq
Content-Type application/json Bəli
x-api-key API_AÇARINIZ Bəli
Body Parametrləri
Açar Növ Açıqlama
recipient string Mesaj göndəriləcək nömrə (məs: 994XXXXXXXXX)
message string Göndəriləcək mesaj mətni
Nümunəvi İstifadə
cURL
curl -X POST http://localhost:2004/api/message \
  -H "Content-Type: application/json" \
  -H "x-api-key: API_AÇARINIZ" \
  -d '{
    "recipient": "994551234567",
    "message": "Salam! HubMSG API vasitəsilə göndərildi."
  }'
Nod.js (axios)
const axios = require('axios');

axios.post('http://localhost:2004/api/message', {
    recipient: '994551234567',
    message: 'Salam! HubMSG API vasitəsilə göndərildi.'
}, {
    headers: {
        'x-api-key': 'API_AÇARINIZ',
        'Content-Type': 'application/json'
    }
}).then(res => console.log(res.data))
  .catch(err => console.error(err));
Xəta Kodları
  • 200 Uğurlu, mesaj növbəyə əlavə edildi.
  • 401 Yanlış və ya çatışmayan API açarı.
  • 400 Çatışmayan parametr (alıcı və ya mesaj boşdur).
  • 500 Server xətası və ya aktiv cihaz tapılmadı.
